## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:9080
$ yarn run dev

# build electron app for production
$ yarn run build

# lint all JS/Vue component files in `app/src`
$ yarn run lint

# lint and fix
$ yarn run lint:fix

# test electron app for production
$ yarn run test
```
